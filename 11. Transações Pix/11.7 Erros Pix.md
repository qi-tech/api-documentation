# Erros Pix

Para testar os possíveis casos de erro, basta iniciar um PIX e usar uma das mensagens na tabela abaixo no campo
**message**.

## Exemplo de Requisição

```json
{
  "pix_transfer_type": "key",
  "message": "AB03",
  "source_account": {
    "account_branch": "0001",
    "account_digit": "3",
    "account_number": "24339",
    "owner_document_number": "32402502000135"
  },
  "pix_key": "qitech@qitech.com.br",
  "transaction_amount": 19000,
  "schedule_date": "2021-08-04",
  "receiver_conciliation_id": "REC00000000000000000000009459463343",
  "requester_document_identification": "11111111111"
}
```

<br>

## Exemplo de Resposta

```json
{
  "title": "SPI Timeout Control",
  "description": "SPI Timeout Control.",
  "translation": "Controle de timeout no SPI.",
  "extra_fields": {
    "description": "SPI Timeout Control.",
    "translation": "Controle de timeout no SPI.",
    "external_title": "SPI Timeout Control",
    "external_code": "PXP000007"
  },
  "code": "PXP000007"
}
```

<br>
